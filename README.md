## 📖 Project Overview

This Digital Literacy Project was completed as part of the CSE0001 course. The aim of this project is to develop essential digital skills such as creating online content, building a professional digital presence, practicing coding, understanding online communication, and spreading awareness about cyber safety.

The project is divided into five tasks, each focusing on a different aspect of digital literacy.
